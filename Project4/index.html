Is Strassen Matrix Multiplication Slower on 'Small' Matrices?

The following 2 graphs depict the function call stacks for a program that runs 2 matrix multiplication implementations against each other, a Strassen matrix multiplication and naive matrix multiplication against each other. The size of the matrices used are 512x512. One uses a switching size of 2 (meaning the Strassen switches to simple multiplication once hitting a 2x2 matrix) while the second graph using a switching size of 128. As seen by the icicle graphs, increasing the size at which the Strassen switches to naive multiplication increases the performance, indicating Strassen is slower on small sized matrices.
